# SOC Week 2 - Alert Management and Incident Response

## Objective

This project demonstrates Security Operations Center (SOC) activities including alert prioritization, incident classification, alert triage, log analysis, and incident response.

## Tools Used

* Wazuh
* Kali Linux
* Linux Logs (/var/log/auth.log)
* iptables

## Steps Performed

1. Created alert priority classification using CVSS scoring
2. Classified incidents using MITRE ATT&CK framework
3. Performed alert triage to identify brute-force attack
4. Monitored alerts using Wazuh dashboard
5. Simulated SSH brute-force attack
6. Analyzed logs using /var/log/auth.log
7. Applied firewall rule using iptables
8. Collected evidence and generated SHA256 hash
9. Followed incident response lifecycle

## Outcome

Successfully detected, analyzed, and mitigated a brute-force attack using SOC methodologies.
